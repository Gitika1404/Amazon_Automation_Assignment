# AmazonAutomation 

## 📌 Overview
This project automates product search and cart functionality on **Amazon.com** using:
- **Java**
- **Selenium WebDriver**
- **TestNG**
- **Maven**

It demonstrates a clean Page Object Model (POM) structure and professional GitHub repo organization.

---

## ⚙️ Setup Instructions
1. **Clone the repository**
   ```bash
   git clone https://github.com/Gitika1404/Amazon_Automation_Assignment.git
   cd Amazon_Automation_Assignment
2. **Install  Dependencies**
    ```bash
    mvn clean install
3. **Run Tests*
    ```bash
    mvn clean test

## 🧪 Features
- Search for products on Amazon.
- Handle dynamic DOM and product variants (e.g., iPhones).
- Extract product prices reliably.
- Add products to cart and confirm cart update.
- Clear console output with short pause after cart confirmation.

## 📄 Notes
- Reports generated by TestNG are excluded via .gitignore (test-output/).
- Repo is recruiter‑ready: clean structure, no build artifacts, clear documentation.
